## 🛠 Quraşdırma

1. Yuxarıdakı **[ParkStore-Setup.exe endir](https://github.com/1CSoftware/ParkStore-Release/releases/latest/download/ParkStore-Setup.exe)** düyməsinə klik et → fayl endirilir
2. Faylı işə sal → quraşdırıcını izlə *(Program Files-ə quraşdırılır + masaüstü qısayolu)*
3. İlk açılışda **brend adını** daxil et → ünvanın avtomatik yaranır: `{brend}.1csoft.az`
4. Admin panelinin **Terminallar** səhifəsindəki **5 rəqəmli parolu** kassaya daxil et
5. RFID oxuyucunu qoş → **hazırdır** ✅

## 💻 Tələblər

- **Windows 10 / 11** (64-bit)
- RFID oxuyucu — **ACR122U** və ya uyğun PC/SC cihaz
- İnternet *(ilk açılışda kart açarları və kassir siyahısı üçün; sonra oflayn işləyir)*

## 🔄 Yeniləmə

Yeni versiya çıxanda **eyni linkdən** son setup-u endir və üstündən quraşdır — konfiqurasiyan (brend, terminal, kassir) qorunur.

## 🔒 Təhlükəsizlik

Bu proqram **heç bir gizli açar saxlamır** — kart açarları yalnız serverdən, autentifikasiya olunmuş terminala verilir və cihazda şifrəli (DPAPI) saxlanılır.
